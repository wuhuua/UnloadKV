## :articulated_lorry: 环境准备

无论在任何环境上编译，请确保你的环境拥有以下工具链：

```shell
(你的环境上应已经安装了make工具)
cmake 版本 >= 3.13
gcc/clang gcc建议8.3以上，编译器需要支持c++20新标准
```


## :thinking: 如何编译:question:

执行下面的命令即可完成编译：

```bash
make
```
